# 🔈 Music Player

A simple application where user can play some music.

## 🚀 Features

- ✅ Play/Pause
- ✅ Seek playtime
- ✅ Displays:
  - music player
  - play/pause button
  - progress bar
  - next/prev button
- ✅ Simple and responsive design using HTML, CSS, and JavaScript
- ✅ Fetches data from stored object

---

## 🛠️ Tech Stack

- **HTML** — Page structure
- **CSS** — Styling and layout
- **JavaScript** — dynamic DOM updates
